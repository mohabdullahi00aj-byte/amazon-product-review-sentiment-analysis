## Overview
This project performs sentiment analysis on Amazon product reviews
to classify reviews as positive or negative using machine learning.

## Dataset
Amazon product review dataset (Excel format).

## Methods
- Text preprocessing (tokenization, stopword removal, lemmatization)
- TF-IDF feature extraction
- Logistic Regression
- Random Forest

## Evaluation
- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC

- Pandas, NumPy
